# Dotfiles

A repo for keeping my dotfiles neat and organized.

## Installation

Create a directory, clone repo and run the installer.

```bash
$ cd ~
$ mkdir .dotfiles && cd .dotfiles
$ git clone https://github.com/jeromverschoote/dotfiles.git .
$ ./install
```
